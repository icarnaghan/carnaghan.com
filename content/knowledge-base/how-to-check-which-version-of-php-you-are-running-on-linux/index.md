- In the command line type in the following command to get allÂ **PHP information and configurations**
    
    ```
    php -i
    ```
    
- To get the Version number use the following command
    
    ```
    php -v
    ```
    
- Or this command:
    
    ```
    php -i | grep 'PHP Version'
    ```
