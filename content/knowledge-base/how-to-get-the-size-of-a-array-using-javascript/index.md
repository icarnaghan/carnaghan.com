date: 2018-03-21
---

To get the **total** **size** of the **array elements** within an **JavaScript** array the **JavaScript**_**length**_ property can be used.

E.g.

```
	var arr = new Array();
 
	arr[0] = 'val 1';
 
	arr[1] = 'val 2';
 
 
 
	alert(arr.length); //Will output 2
```
