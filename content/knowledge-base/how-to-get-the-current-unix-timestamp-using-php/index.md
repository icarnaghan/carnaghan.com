_**UNIX TIMESTAMP**_: _Is the time measured in seconds since the start of the Unix epoch "1 January 1970"_

Use the **PHP time** function as per example below

```
echo time(); //Will print out the current timestamp
 
//OUTPUT: 1285365600
```
