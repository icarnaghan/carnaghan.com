date: 2018-04-07

Run the following query: **SELECT version();**

The result will look as below:

                                               **version --------------------------------------------------------------------------------------------------- PostgreSQL 8.3.7 on i486-pc-linux-gnu, compiled by GCC gcc-4.3.real (Ubuntu 4.3.3-5ubuntu4) 4.3.3 (1 row)**
