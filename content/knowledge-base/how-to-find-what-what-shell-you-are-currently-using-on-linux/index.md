title: "How to find what what shell you are currently using on Linux"
date: 2018-04-07
---

- Open a new **Terminal Window** 
- Type in the following command
    
    ```
    echo $SHELL
    ```
    
- This will output something similar like below
    
    ```
    root@local-laptop:~$ echo $SHELL
    /bin/bash
    ```
